Varios ficheros abiertos al mismo tiempo        (08/Oct/20)
¿Hacerlo con MDI? (sería lo más fácil)
    poniendo los menús y barras de herramientas en el formulario MDI
    y abriendo un Form1 para cada fichero.

Sin MDI (sería más laborioso, al menos para mostrar las ventanas abiertas)
    el primer Form1 asignaría el valor a una variable compartida (al estilo de Current)
    con idea de que el resto sepa quién es el principal que sería el que gestionaría
    el menú Ventanas con los demás formularios cargados.
    La ventaja es que serían independientes y no se mostarían todos dentro del MDI,
    si no como ventanas sueltas.
    Cuando se cierre el primero que se abrió se cierran todos.
    O incluso poder cerrar el resto y dejar solo el primero.
    Esto vendría bien por si se abre un proyecto.

    Otra opción es abrir cada formulario en TABS dentro del primero.
    Pero usando valores independientes para el nombre del fichero, los bookmarks, etc.
    En realidad no se abrirían los formularios si no el editor y los números de líneas.
    Ya que la lista de sintaxis podría valer para todos... por si se compila el proyecto.

    De esa forma se podrían tener varias instancias con proyectos separados...

    La idea es poder abrir también una solución con varios proyectos.
